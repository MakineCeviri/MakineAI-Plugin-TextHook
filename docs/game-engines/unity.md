# Unity Desteği

Unity motor dosya yapısı ve çeviri yöntemi teknik referansı.

> **Not:** Bu doküman teknik referanstır. Makine-Launcher şu an motor bazlı işlem yapmamaktadır — sadece motor tespiti yapar. Bu bilgiler adaptasyon motoru tasarımında kullanılacaktır.

---

## Genel Bakış

**Destek Durumu:** Tam Destek

**Desteklenen Sürümler:**
- Unity 2018.x - 2023.x
- Mono build
- IL2CPP build

---

## Nasıl Çalışır

Makine-Launcher Unity oyunlarında BepInEx + XUnity.AutoTranslator kullanır:

1. **BepInEx** - Unity mod framework
2. **XUnity.AutoTranslator** - Çeviri hook sistemi

### Çalışma Prensibi

```
Oyun Baslat
    |
    v
BepInEx Yukle
    |
    v
XUnity.AutoTranslator Baslat
    |
    v
Metin Hooking
    |
    v
Ceviri Dosyasi Yukle
    |
    v
Turkce Metin Goster
```

---

## Otomatik Algılama

Makine-Launcher Unity oyunlarını şu dosyalardan tespit eder:

| Dosya | Açıklama |
|-------|----------|
| `UnityPlayer.dll` | Unity runtime |
| `GameName_Data/` | Data klasörü |
| `Assembly-CSharp.dll` | Mono build |
| `GameAssembly.dll` | IL2CPP build |
| `globalgamemanagers` | Unity asset |

## Çeviri Süreci

### 1. BepInEx Kurulumu

```
[Oyun Klasoru]/
├── BepInEx/
│   ├── core/
│   ├── plugins/
│   └── config/
├── doorstop_config.ini
└── winhttp.dll
```

### 2. XUnity Kurulumu

```
[Oyun Klasoru]/
└── BepInEx/
    └── plugins/
        └── XUnity.AutoTranslator/
            ├── XUnity.AutoTranslator.dll
            └── Translation/
                └── tr/
                    └── Text/
                        └── _AutoGeneratedTranslations.txt
```

### 3. Çeviri Dosyası

Format: Key=Value

```
Hello World=Merhaba Dunya
Start Game=Oyuna Basla
Settings=Ayarlar
```

## Teknik Detaylar

### Mono Build

- IL kodu runtime'da yakalanır
- UGUI Text, TextMeshPro desteği
- Dinamik çeviri mümkün

### IL2CPP Build

- Native hook kullanılır
- Performans daha iyi
- Bazı limitasyonlar olabilir

### Font Desteği

XUnity otomatik font değişimi yapar:
- Unity default fontlar
- TextMeshPro fontlar
- Türkçe karakterler için uygun font

## Bilinen Sorunlar

### Anti-Cheat

Bazı Unity oyunları anti-cheat kullanıyor:
- EasyAntiCheat
- BattlEye

**Çözüm:** Sadece singleplayer modda kullanın

### IL2CPP Sınırları

IL2CPP build'lerde:
- Bazı reflection hook'lar çalışmayabilir
- Daha az dinamik yetenek

### Asset Bundle Metinleri

Asset bundle içindeki metinler:
- XUnity ile yakalanamayabilir
- Ayrı asset patch gerekebilir

## Troubleshooting

### BepInEx Yüklenmiyor

1. `doorstop_config.ini` kontrol et
2. `winhttp.dll` varlığını kontrol et
3. Anti-virus istisna ekle

### Çeviri Görünmüyor

1. `_AutoGeneratedTranslations.txt` kontrol et
2. Encoding UTF-8 olmalı
3. XUnity config'de dil "tr" olmalı

### Oyun Çöküyor

1. BepInEx sürüm uyumluluğu kontrol et
2. Diğer mod'larla çakışma olabilir
3. Log dosyasını kontrol et: `BepInEx/LogOutput.log`
